## 📚 課程目的
- 保存課程所有的程式作業練習
- 學習使用 GitHub
- 高效學習資料結構

> Tip: 將每次作業附上「題目連結 / 解題思路 / 複雜度分析」。

## 📂 資料夾結構
| 資料夾 | 說明 |
|---|---|
| `課堂作業/` | 每週指定作業，含測試與報告 |
| `範例程式碼/` | 教材範例與延伸實作 |
| `課堂筆記/` | 重點整理、錯誤案例與心得 |

---

## 🧩 技術重點
- **資料結構主軸：** Array、Linked List、Stack、Queue、Tree、Graph
- **演算法練習：** Sorting、Searching、Traversal、Greedy、DP
- **工具：** VS Code、Git、C++/Python、Catch2 或 GoogleTest

### ✅ 作業提交規範
- **命名：** `hw{週次}/{題目}/{你的學號}_{姓名}.cpp`
- **內容：** 題目、思路、時間/空間複雜度、測試案例
- **測試：** 提供至少 5 個邊界案例
